# Recruitment Process

The general recruitment process is:

```text
application form + machine test + interview
```

## Application Form

The form collects basic information, interests, experience, and availability. It should not ask applicants to disclose unnecessary private information.

## Machine Test

The machine test checks basic programming, problem solving, documentation, and responsible use of tools. Real test materials are stored privately and are not published here.

## Interview

The interview focuses on motivation, learning path, technical reasoning, collaboration, and understanding of submitted work.

## Results

Results should be communicated respectfully. Candidate evaluations and internal notes are private records.
