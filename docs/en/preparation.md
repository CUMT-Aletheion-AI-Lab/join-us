# Preparation Suggestions

Applicants can prepare through:

- learning basic Git and GitHub workflow;
- practicing Python and command-line usage;
- reading beginner-friendly AI materials;
- documenting experiments and decisions;
- learning how to ask clear technical questions;
- making small, focused contributions to open-source projects.

## What We Value

- curiosity and persistence;
- technical honesty;
- ability to explain your own work;
- willingness to collaborate;
- responsible use of AI tools.

## What Not to Do

Do not try to memorize leaked answers or submit work you cannot explain. This reduces evaluation reliability and does not help long-term learning.
